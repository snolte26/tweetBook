tweetBook

A Bot to interact with the tweetBook SMS


About

First created and uploaded to GitHub early March 2021, tweetBook is the passion project of Alexander Nolte, with the hopes of creating a small and private SMS. Currently it runs as a command line application, and users can create their own accounts, add posts, search users, view their posts, and delete posts. All posts are deleted after 30 days thanks to the AutoMod, and user login information is stored locally. There are no ads or trackers on tweetBook

With the tweetBook discord bot, users can now anonymously send and view messages from tweetBook, while protecting the identity of who sent the message.

Commands

  • tweetBookAdd
    • To send a message to tweetBook, simply type "tweetBookAdd" followed by whatever message you wish to send. It must all be in a single message
  • tweetBookRefresh
    • Will send all of the posts within the tweetBook server. Depending on how many posts there are, this action can take forever
  • %help
    • Will bring up this site for those that need help or need to know the commands.
